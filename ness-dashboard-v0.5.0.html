RPC: 6662 (cookie auth)

Emercoin JSON-RPC on 6662 uses a cookie for auth inside the container, which browsers cannot access safely. For local control, prefer the ./emercoin-cli wrapper or a trusted Emercoin GraphQL endpoint at Emercoin GraphQL endpoint.
